# 视频生成 Skill

## 概述

使用视频生成 API 或工具创建视频内容。

## 工作流程

1. 确认用户需求：视频主题、时长、风格、分辨率、用途
2. 准备脚本或分镜描述
3. 调用视频生成接口
4. 保存结果到 `E:\CODEX\videos\` 目录
5. 按项目命名规则保存：`项目名称_日期_序号.mp4`

## 注意事项

- 遵循 `E:\CODEX\AGENTS.md` 的文件分类存储规范
- 生成的视频保存到 `E:\CODEX\videos\`
- 新项目视频放入 `E:\CODEX\projects\项目名称_日期\` 子目录
- 视频生成可能需要较长时间，及时告知用户进度
- 注意时长和分辨率的限制
