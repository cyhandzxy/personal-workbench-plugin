# 图片生成 Skill

## 概述

使用 OpenAI DALL·E 或其他图像生成 API 创建高质量图片。

## 工作流程

1. 确认用户需求：图片主题、风格、尺寸、用途
2. 优化 prompt，确保描述清晰具体
3. 调用图像生成接口
4. 保存结果到 `E:\CODEX\images\` 目录
5. 按项目命名规则保存：`项目名称_日期_序号.png`

## 注意事项

- 遵循 `E:\CODEX\AGENTS.md` 的文件分类存储规范
- 生成的图片保存到 `E:\CODEX\images\`
- 新项目图片放入 `E:\CODEX\projects\项目名称_日期\` 子目录
- 保持 prompt 简洁但信息充分
- 如需编辑已有图片，明确修改范围
